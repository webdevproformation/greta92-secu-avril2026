# injection XSS

XSS Cross Site Scripting

Injection de Javascript 

- 3 variantes d'injection XSS (+ 1 bonus)
- 
- XSS stored (type1) => une fois que l'injection est réussie => le javascript va être stocké : base de données / localStorage / même dans les cookies
- XSS Reflected (refléchie / non stockée) type 2 : l'injection js fonctionne MAIS pas stockée 
- XSS DOM => type d'attaque sur les nouvelles manières de créer des sites internet :  React / Angular / Vue
- Self XSS : javascript que vous allez vous auto infliger
