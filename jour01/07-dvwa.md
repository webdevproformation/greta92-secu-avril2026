# installation du projet dvwa sur votre ordinateur 

1. `docker -v` # vérifier que docker est bien installé sur l'ordinateur
1. `sudo systemctl start docker`
1. <https://github.com/digininja/DVWA>
1. télécharger le zip
1. déplacer le zip dans le dossier secu-web
1. extraire le zip
1. lancer une terminal dans Visual Studio
1. cd DVWA / ou cd DVWA-master
1. docker compose up -d --build
